## 📚 学习内容

### 主题：反调试与反混淆技术

**来源**: Al-Khaser 项目 v0.81 + 综合知识库

**学习模块**:

1. **反调试技术** (Anti-Debugging)
   - API 检测法 (IsDebuggerPresent, CheckRemoteDebuggerPresent)
   - PEB 结构检测 (BeingDebugged, NtGlobalFlag, Heap Flags)
   - Native API 检测 (NtQueryInformationProcess)
   - 异常处理检测 (UnhandledExceptionFilter, SEH)
   - 时间检测 (RDTSC, Sleep 加速)
   - 硬件特征检测 (INT 2D, INT 3, 陷阱标志)
   - 进程环境检测 (父进程, 调试特权)

2. **反虚拟机技术** (Anti-VM)
   - 注册表检测 (VMware, VirtualBox 特征键)
   - 文件系统检测 (VM 驱动文件)
   - CPU 指令检测 (CPUID Hypervisor 检测)
   - MAC 地址检测 (VM 网卡前缀)
   - 内存特征检测 (IDT/GDT/LDT 位置)

3. **反沙箱技术** (Anti-Sandbox)
   - 环境检测 (内存、磁盘、CPU 核心数)
   - 人机交互检测 (鼠标移动)
   - 用户名/主机名检测

4. **反混淆技术** (Anti-Obfuscation)
   - 常见混淆方法 (控制流、数据、代码混淆)
   - 去混淆方法 (静态/动态分析)

5. **绕过技术**
   - 反调试绕过 (修补代码、Hook、插件)
   - 反 VM 绕过 (修改特征、Hook CPUID、裸机分析)
   - 反沙箱绕过 (等待输入、执行延迟、环境伪装)
