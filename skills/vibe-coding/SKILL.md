## What is Vibe Coding?

Vibe Coding (æ°›å›´ç¼–ç¨‹) is an emerging software development practice that uses AI to generate functional code from natural language prompts. Coined by Andrej Karpathy, it transforms programming from "writing code" to "conversational creation".

## Core Philosophy

- **From Coder to Commander**: Transform from writing code to directing AI
- **95%+ AI Implementation**: AI handles implementation, you focus on creativity
- **Natural Language Programming**: Use everyday language to describe what you want

## Best Practices

1. **Be Specific**: Describe exactly what you want
2. **Iterate**: Refine prompts based on results
3. **Verify**: Always test AI-generated code
4. **Document**: Save what works for reuse
5. **Learn**: Treat each session as a learning opportunity
