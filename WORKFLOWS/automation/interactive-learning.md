# 工作流: 交互学习

## 描述
在对话中实时学习新知识

## 输入
- 主题: string, 必填
- 深度: string, 可选 (basic/intermediate/advanced)

## 触发方式

### 用户触发
```
"学习 React Hooks"
"了解量化交易"
"学习 x86 汇编基础"
```

### 自动触发
- 遇到不熟悉的概念
- 检测到知识盲区
- 用户问题超出当前知识范围

## 步骤

### 1. 主题分析
- 动作: 确定学习领域和深度
- 工具: NLP 分析
- 输出: 学习计划

### 2. 资源搜索
- 动作: 搜索最佳学习资源
- 工具: Tavily Search
- 输出: 资源列表

### 3. 深度学习
- 动作: 阅读并理解核心内容
- 工具: web_fetch + LLM
- 输出: 知识摘要

### 4. 实践验证
- 动作: 生成练习题或代码
- 工具: 代码执行
- 输出: 实践结果

### 5. 知识存储
- 动作: 写入知识库
- 工具: write
- 路径: memory/knowledge/[领域]/[主题].md

### 6. 反馈确认
- 动作: 确认理解程度
- 工具: 对话
- 输出: 学习完成确认

## 输出
- 知识文件: memory/knowledge/
- 理解度: 1-10 分
- 后续建议: 相关主题

## 学习模板

### 基础 (basic)
```
1. 概念定义
2. 核心原理
3. 简单示例
4. 常见应用
```

### 进阶 (intermediate)
```
1. 深入原理
2. 最佳实践
3. 性能考量
4. 实际项目
```

### 高级 (advanced)
```
1. 源码分析
2. 边界情况
3. 优化技巧
4. 前沿发展
```
