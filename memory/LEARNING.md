# LEARNING.md - 自主学习规则

> 定义 Claw 如何从交互中自主学习和积累知识。

## 学习触发器

自动学习以下类型的信息：

### 1. 用户偏好
- 沟通风格调整
- 常用工具/命令
- 工作流程偏好
- 时间习惯

### 2. 技术知识
- 命令行技巧
- 代码片段
- 故障排除经验
- 工具使用心得

### 3. 项目上下文
- 项目结构
- 重要决策
- 进行中的任务
- 依赖关系

### 4. 人际网络
- 常联系的人
- 团队结构
- 角色职责

## 学习行为

### 主动学习
- 从对话中识别重要信息 → 自动提取到 MEMORY.md
- 发现用户模式 → 记录到 memory/preferences.md
- 解决问题后 → 记录解决方案到 memory/solutions/

### 定期整理
- 每日日志汇总 → 更新 MEMORY.md
- 清理过时信息
- 合并重复条目

## 知识分类

```
memory/
├── 2026-02-16.md      # 每日日志
├── preferences.md     # 用户偏好
├── solutions/         # 问题解决方案
├── skills/            # 学到的技能
└── projects/          # 项目相关知识
```

## 学习原则

1. **不打扰** - 静默学习，不打断对话
2. **验证后存储** - 确认重要信息再永久记录
3. **定期回顾** - 避免信息过时
4. **隐私优先** - 不记录敏感信息（密码、密钥等）
