### 📖 学习内容

#### 1. Ghidra 概述
- **定义**：NSA 开发的免费开源逆向工程框架
- **特点**：
  - 跨平台（Windows/Linux/macOS）
  - 高度可扩展的插件系统
  - 支持团队协作
  - 内置撤销功能

#### 2. 核心架构（五大组件）
- **Programs**：数据存储在自定义数据库
- **Plugins**：功能模块，可动态加载
- **Tools**：插件集合，可自定义配置
- **Project Manager**：项目管理器
- **Ghidra Server**：支持多用户协作

#### 3. 基本工作流程
1. 创建项目（共享/非共享）
2. 导入程序（PE/ELF/RAW 等）
3. 自动分析（反汇编/函数识别/交叉引用）
4. 使用 CodeBrowser 查看：
   - Program Tree
   - Symbol Tree
   - Decompiler
   - Listing

#### 4. 分析策略
- **分层分析方法**：
  - 初始：基本分析（关闭推测性选项）
  - 后续：针对性分析（One-Shot 分析器）
- **问题修复**：使用脚本修复分析错误
