### ✅ 并发编程 - Node.js/Python (12:00)
**学习时长**: ~30分钟
**来源**: 空闲自动学习任务

**核心收获**:
1. **并发 vs 并行**
   - 并行: 同时执行多个操作 (CPU密集型)
   - 并发: 任务交替执行 (I/O密集型)
   - 异步I/O性能提升可达 12-24 倍

2. **Python asyncio**
   - 核心语法: `async def`, `await`
   - 事件循环: `asyncio.run()`, `asyncio.gather()`
   - 避免阻塞: 用 `asyncio.sleep()` 替代 `time.sleep()`

3. **JavaScript async/await**
   - 自动返回 Promise
   - 错误处理: try/catch 或 .catch()
   - 并行执行: `Promise.all()`

4. **常见陷阱**
   - 忘记 await
   - 使用阻塞调用阻塞事件循环
   - 串行执行本应并行的任务

**关键代码模式**:
```python
# Python 并发执行
await asyncio.gather(task1(), task2(), task3())

# JavaScript 并行执行
let [a, b] = await Promise.all([fetch(url1), fetch(url2)]);
```

**适用场景**:
- ✅ 网络请求、数据库查询、文件I/O
- ❌ CPU密集计算、紧密循环
