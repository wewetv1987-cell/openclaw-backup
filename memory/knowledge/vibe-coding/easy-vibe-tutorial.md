# Easy-Vibe 教程学习笔记

> 来源：https://github.com/datawhalechina/easy-vibe
> 学习时间：2026-02-17

## 🎯 核心理念

**"在 AI 时代，把想法变成产品的人，往往技术不是最强，而是最先迈出行动。"**

Easy-Vibe 的目标：假设每个人都是零基础，手把手带你从写出第一行代码，到理解前后端逻辑，最后把产品上线。

### 面向人群
- 初学者
- 产品经理
- 前后端/全栈开发者

### 主题领域
- AI 编程
- 全栈 Web 应用开发
- AI Agent
- 工作流和 RAG 系统

## 💡 实践要点

### 新手入门建议
1. 从游戏案例开始（贪吃蛇等）
2. 使用 AI IDE 工具（Cursor、Trae 等）
3. 先做原型，再迭代完善
4. 接入 AI 能力增强产品

### 全栈开发流程
```
设计(Figma) → 前端组件 → API设计 → 数据库 → 部署 → 收费
```

### 多平台开发
- 小程序：微信生态
- 安卓/iOS：Expo 跨平台方案
- Web：现代前端组件库

### AI 能力集成
- 文本生成
- 图像处理
- 知识库 RAG
- 工作流编排

## 🚀 学习建议

### 本地启动课件
```bash
npm install
npm run dev
# 访问 http://localhost:3000
```

或使用 AI IDE 提示词：
```
请你帮我运行这个项目的本地服务
```

### 学习路径选择
- **零基础**：从第一阶段开始
- **有前端经验**：直接进入后端部分
- **全栈开发者**：学习高级章节
