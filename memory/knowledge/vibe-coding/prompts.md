# Vibe Coding 高效提示词集合

## 入门级提示词

### 快速开始

```
Help me create a simple [project type] with [tech stack]
```

### 理解需求

```
I want to build [what you want]
Here are the requirements:
- [requirement 1]
- [requirement 2]

What's the best approach?
```

---

## 项目创建

### Web 应用

```
Create a [app type] web application:
Tech stack: React + TypeScript + Tailwind CSS

Features:
- [feature 1]
- [feature 2]
- [feature 3]

Additional requirements:
- Responsive design
- Error handling
- Loading states
```

### API 服务

```
Build a REST API for [domain]:
Backend: Node.js + Express + MongoDB

Endpoints:
- GET /api/[resource] - list all
- GET /api/[resource]/:id - get single
- POST /api/[resource] - create
- PUT /api/[resource]/:id - update
- DELETE /api/[resource]/:id - delete

Include:
- Input validation
- Error handling
- Authentication (if needed)
```

### 全栈应用

```
Create a full-stack [app type]:
Frontend: React + [additional libraries]
Backend: Node.js + Express
Database: [MongoDB/PostgreSQL]

Core features:
1. [feature category]
   - [sub-feature 1]
   - [sub-feature 2]

2. [feature category]
   - [sub-feature 1]
   - [sub-feature 2]

Include:
- User authentication
- Authorization
- Error handling
- Logging
```

## 提示词技巧

### 1. 提供上下文
```
I'm working on [project type].
Current progress: [brief description]
```

### 2. 明确约束
```
Constraints:
- Use [specific library]
- Keep it [simple/fast/etc]
- File size limit: [limit]
```

### 3. 分步骤
```
Help me with this in steps:
1. First step
2. Second step
3. Third step
```

### 4. 要求解释
```
Show me the code and explain:
- Why this approach
- How it works
- Alternatives considered
```

### 5. 迭代优化
```
This is close but needs adjustment:
[code]

Fix [specific issues]
```
