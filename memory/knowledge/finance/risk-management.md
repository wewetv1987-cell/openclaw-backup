# 金融风险管理 (Financial Risk Management)

> 学习日期: 2026-02-17
> 状态: ✅ 已完成基础学习

---

## 📌 核心定义

**金融风险管理**是通过管理金融风险敞口来保护企业经济价值的实践。主要包括：
- 识别风险来源
- 量化风险
- 制定缓解计划

## 🎯 风险类型

### 1. 市场风险 (Market Risk)
- **定义**: 因市场价格变动导致的损失风险
- **包括**: 利率风险、汇率风险、股票价格风险、商品价格风险
- **管理工具**: 对冲 (Hedging)、衍生品

### 2. 信用风险 (Credit Risk)
- **定义**: 交易对手未能履行义务的风险
- **包括**: 违约风险、交易对手信用风险
- **缓解措施**: 保证金 (Margin)、抵押品 (Collateral)、净额结算

### 3. 操作风险 (Operational Risk)
- **定义**: 内部流程、人员、系统或外部事件导致损失的风险
- **特点**: 金融机构的副产品，需控制
- **管理**: 内控流程、系统审计

### 4. 流动性风险 (Liquidity Risk)
- **定义**: 无法及时变现资产或融资的风险
- **指标**: LCR (流动性覆盖率)、NSFR (净稳定资金比率)

## 📊 风险量化工具

### Value at Risk (VaR) - 风险价值
```
定义: 在给定置信度和时间范围内，投资组合可能遭受的最大损失
常用参数:
- 置信度: 95%, 99%
- 时间范围: 1天, 10天, 2周
```

**计算方法**:
1. **参数法** - 假设正态分布
2. **历史模拟法** - 使用历史数据
3. **蒙特卡洛模拟** - 随机模拟

**扩展指标**:
- CVaR (条件风险价值) / Expected Shortfall
- Tail VaR (尾部风险价值)
- Extreme Value Theory (极值理论)

### Greeks - 敏感性指标
用于衡量衍生品价格对各种因素的敏感性：

| Greek | 衡量内容 | 应用 |
|-------|---------|------|
| Delta | 价格对标的价格变化 | 方向性风险 |
| Gamma | Delta的变化率 | 曲率风险 |
| Vega | 价格对波动率变化 | 波动率风险 |
| Theta | 价格对时间衰减 | 时间价值 |
| Rho | 价格对利率变化 | 利率风险 |

**其他敏感度指标**:
- **DV01**: 债券/互换对利率的敏感度
- **CS01**: 对信用利差的敏感度
- **JTD**: 跳跃至违约风险

### 经济资本 vs 监管资本

| 类型 | 定义 | 用途 |
|-----|------|------|
| **经济资本** | 银行自身评估所需资本 | 内部风险管理 |
| **监管资本** | 监管机构要求持有的资本 | 合规要求 |

## 🏦 不同机构的风险管理重点

### 投资银行
- **重点**: 市场风险、信用风险
- **业务**: 自营交易、做市、结构化产品
- **工具**: Greeks、VaR、压力测试

### 商业/零售银行
- **重点**: 信用风险（贷款违约）、流动性风险
- **业务**: 存贷利差
- **关注**: 净利息边际 (NIM)、存款提取风险

### 保险公司
- **寿险**: 长寿风险、利率风险
- **财险**: 巨灾风险、理赔波动性
- **目标**: 偿付能力、理赔支付能力

### 非金融企业
- **重点**: 业务风险（生产、营销）
- **次要**: 市场、信用风险
- **目标**: 保护现金流、降低财务困境概率

## 📜 巴塞尔协议 (Basel Accords)

国际银行业风险管理的监管框架：

### Basel II 三大支柱
1. **支柱 I**: 最低资本要求
   - 信用风险、市场风险、操作风险
   - 标准法或内部模型法 (IRB)
   
2. **支柱 II**: 监管审查过程
   - ICAAP (内部资本 adequacy 评估)
   - 捕捉所有实质性风险
   
3. **支柱 III**: 市场纪律
   - 信息披露要求

### Basel III 改进 (2008危机后)
- **资本要求**: 更高的资本充足率 (至少 12.9% RWA)
- **杠杆率 (LR)**: 限制过度杠杆
- **流动性指标**: LCR, NSFR
- **压力测试**: 更强调银行压力测试

## 🔄 风险管理流程

```
1. 风险识别 → 识别所有潜在风险
2. 风险度量 → 量化风险敞口 (VaR, Greeks)
3. 风险评估 → 评估影响和可能性
4. 风险缓解 → 对冲、分散、转移
5. 风险监控 → 持续监控和报告
6. 风险审查 → 定期审查和改进
```

## 🛡️ 风险缓解策略

### 1. 对冲 (Hedging)
- 使用衍生品抵消风险
- 工具: 期货、期权、互换
- 注意: 对冲成本 vs 收益

### 2. 分散化 (Diversification)
- "不要把鸡蛋放在一个篮子里"
- 降低非系统性风险
- 现代投资组合理论 (Markowitz, 1952)

### 3. 风险转移
- 购买保险
- 证券化
- 信用衍生品 (CDS)

### 4. 风险规避
- 不参与高风险活动
- 退出特定业务线

## 📈 2008金融危机的教训

### 失败原因
1. **模型缺陷**: VaR低估尾部风险
2. **系统性风险**: 机构间关联度过高
3. **流动性枯竭**: 市场流动性突然消失
4. **杠杆过高**: 过度使用杠杆
5. **监管不足**: 影子银行系统缺乏监管

### 改进措施
1. **模型进化**:
   - 纳入波动率曲面
   - 多曲线框架
   - CVA/XVA 估值调整
   
2. **监管强化**:
   - 更严格的资本要求
   - 杠杆率限制
   - 流动性缓冲

3. **压力测试**:
   - 定期压力测试
   - 极端情景分析

## 💡 实践要点

### 对于投资者
1. **理解自己的风险承受能力**
2. **分散投资组合**
3. **使用止损策略**
4. **定期再平衡**
5. **不要过度杠杆**

### 对于企业
1. **识别核心业务风险 vs 金融风险**
2. **只对冲投资者无法自行对冲的风险**
3. **对冲成本不应超过潜在损失**
4. **建立风险文化**
5. **定期审查风险管理策略**

## 🎓 关键概念总结

| 概念 | 要点 |
|-----|------|
| **三道防线** | 业务线、风险管理、审计 |
| **风险资本** | 最后一道防线 |
| **对冲无关性命题** | 完美市场中对冲不创造价值 |
| **Basel协议** | 银行业风险管理国际标准 |
| **VaR** | 最广泛使用的风险指标 |
| **压力测试** | 评估极端情况下的损失 |
