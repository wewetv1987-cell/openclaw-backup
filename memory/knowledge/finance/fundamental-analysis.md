# 基本面分析 (Fundamental Analysis)

> 学习时间: 2026-02-17 10:27
> 来源: Wikipedia, 金融基础知识整理

## 📖 核心定义

基本面分析是分析企业财务报表的方法，用于评估企业的：
- 资产与负债
- 盈利能力
- 健康状况
- 竞争对手和市场环境

同时考虑整体经济状态：利率、生产、就业、GDP、住房、制造业和管理层。

## 🎯 分析目标

1. **股票估值** - 预测股价走势
2. **业绩预测** - 评估企业经营表现
3. **管理评估** - 做出内部决策或计算信用风险
4. **内在价值** - 发现股票的真实价值

## 🔬 三层分析框架

### 1. 经济分析 (Economic Analysis)
- GDP 增长率
- 通胀率
- 利率
- 汇率
- 生产力指标
- 能源价格

### 2. 行业分析 (Industry Analysis)
- 总销售额
- 价格水平
- 竞争产品影响
- 外国竞争
- 行业进入/退出壁垒

### 3. 公司分析 (Company Analysis)
- 财务报表分析
- 财务比率
- 股息支付
- 经营现金流
- 新股发行和资本融资

## 📊 投资策略应用

### 买入持有 (Buy and Hold)
- 寻找"好公司"，降低风险
- 让资产随企业成长

### 价值投资 (Value Investing)
- 关注被低估的公司
- "很难从沟里掉下去"

### 逆向投资 (Contrarian)
- "短期内市场是投票机，不是称重机"
- 独立判断价值，忽略市场噪音

## 🔄 两种分析方法

### 自上而下 (Top-Down)
```
全球经济 → 国家经济 → 行业分析 → 最佳公司
```
从宏观到微观，层层筛选

### 自下而上 (Bottom-Up)
```
特定公司 → 行业背景 → 经济环境
```
从具体公司出发，不拘泥于行业

## 💰 估值模型

### 折现现金流模型 (DCF)
计算未来现金流的现值：
- 股息折现 (Gordon 模型)
- 收益折现
- 自由现金流折现

### 关键参数
- 增长率 (收入和现金流)
- 风险水平 (确定折现率)

## 🆚 基本面 vs 技术面

| 维度 | 基本面分析 | 技术面分析 |
|------|----------|----------|
| 理念 | 市场短期可能错误定价，长期会修正 | 价格趋势反映市场情绪 |
| 关注 | 公司内在价值 | 历史价格走势 |
| 时间 | 中长期投资 | 短期交易 |
| 工具 | 财务报表、经济数据 | K线图、技术指标 |

**实践中可以结合使用：**
- 基本面选股（找好公司）
- 技术面择时（决定买卖点）

## 📝 决策逻辑

```
如果 内在价值 > 市场价格 → 买入
如果 内在价值 = 市场价格 → 持有
如果 内在价值 < 市场价格 → 卖出
```

## 🔑 关键指标速查

| 指标 | 说明 | 用途 |
|------|------|------|
| P/E (市盈率) | 股价/每股收益 | 估值水平 |
| P/B (市净率) | 股价/每股净资产 | 资产价值 |
| ROE | 净利润/股东权益 | 盈利效率 |
| Debt/Equity | 总负债/股东权益 | 财务风险 |
| Free Cash Flow | 经营现金流 - 资本支出 | 真实盈利能力 |
| Dividend Yield | 每股股息/股价 | 分红回报 |
