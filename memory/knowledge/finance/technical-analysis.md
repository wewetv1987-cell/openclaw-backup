# 技术分析指标 (Technical Analysis Indicators)

> 学习时间: 2026-02-17 06:30
> 来源: Wikipedia, BabyPips

---

## 📖 概述

技术分析是通过研究历史市场数据（主要是价格和成交量）来分析和预测价格走势的分析方法。

### 核心哲学
- **市场效率**: 价格反映所有公开信息
- **群体心理**: 市场参与者的情绪和行为创造模式和趋势
- **历史会重演**: 人性和市场动态导致类似的模式和周期
- **概率思维**: 技术分析处理的是概率，而非确定性

### 三大假设
1. **市场行为包容一切** - 所有相关信息已反映在价格中
2. **价格以趋势方式演变** - 价格方向性移动：上涨、下跌或横盘
3. **历史会重演** - 价格形态反复出现

## 📊 主要技术指标

### 1. 移动平均线 (Moving Average, MA)

移动平均线是最基础的趋势跟踪指标，用于平滑短期波动并识别趋势方向。

#### 简单移动平均线 (SMA)
```
SMA = (P1 + P2 + ... + Pn) / n
```
- **特点**: 对所有数据点赋予相同权重
- **常用周期**: 20日、50日、200日
- **优点**: 计算简单，稳定性好
- **缺点**: 滞后性强，对旧数据过度敏感

#### 指数移动平均线 (EMA)
```
EMA = α × 当前价格 + (1-α) × 前一日EMA
α = 2 / (n + 1)
```
- **特点**: 对近期价格赋予更大权重
- **常用周期**: 12日、26日
- **优点**: 反应更快，减少滞后
- **缺点**: 可能产生更多假信号

#### 交易信号
- **金叉**: 短期MA上穿长期MA → 买入信号
- **死叉**: 短期MA下穿长期MA → 卖出信号
- **支撑/阻力**: 价格在MA上方/下方

---

### 2. 相对强弱指数 (RSI)

RSI 是一个动量振荡指标，衡量价格变动的速度和幅度。

#### 计算公式
```
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - (100 / (1 + RS))
```

#### 参数配置
- **默认周期**: 14天
- **范围**: 0-100
- **超买区**: > 70（强超买可用80）
- **超卖区**: < 30（强超卖可用20）

#### 交易信号
| 信号 | 描述 |
|------|------|
| **超买** | RSI > 70，价格可能回调 |
| **超卖** | RSI < 30，价格可能反弹 |
| **看跌背离** | 价格创新高但RSI未创新高 → 顶部信号 |
| **看涨背离** | 价格创新低但RSI未创新低 → 底部信号 |
| **中轴线** | RSI = 50，无明确趋势 |

#### 趋势确认 (Cardwell理论)
- **上升趋势**: RSI通常在40-80之间
- **下降趋势**: RSI通常在20-60之间

---

### 3. MACD (移动平均线收敛/发散指标)

MACD 由 Gerald Appel 在1970年代末创建，用于揭示趋势的强度、方向、动量和持续时间。

#### 组成部分
```
MACD线 = 12日EMA - 26日EMA
信号线 = MACD线的9日EMA
柱状图 = MACD线 - 信号线
```

#### 标准设置: MACD(12, 26, 9)
- **12日EMA**: 快线（约2周）
- **26日EMA**: 慢线（约1个月）
- **9日EMA**: 信号线

#### 交易信号
| 信号 | 描述 |
|------|------|
| **金叉** | MACD线上穿信号线 → 买入 |
| **死叉** | MACD线下穿信号线 → 卖出 |
| **零轴穿越** | MACD上穿零轴 → 趋势转强 |
| **背离** | 价格与MACD走势不一致 → 趋势反转信号 |
| **柱状图** | 正值扩大 → 上涨动能增强 |

#### 特点
- **滞后指标**: 基于移动平均线，有延迟
- **趋势市场有效**: 横盘市场效果较差
- **可测量**: 一阶导数（速度）和二阶导数（加速度）

## 📈 图表形态

### 反转形态
- **头肩顶/底** (Head and Shoulders)
- **双顶/双底** (Double Top/Bottom)
- **圆弧底** (Rounding Bottom)

### 持续形态
- **旗形** (Flag)
- **三角形** (Triangles)
- **楔形** (Wedge)

### 支撑与阻力
- 历史价格水平
- 整数关口
- 移动平均线
- **角色互换**: 突破的阻力变支撑，反之亦然

## 🔧 实践要点

### 指标组合使用
```
建议组合:
1. 趋势指标 (MA) + 动量指标 (RSI)
2. MACD + 支撑阻力
3. 多时间框架分析
```

### 风险管理
- 永远设置止损
- 不要过度依赖单一指标
- 结合基本面分析
- 保持交易日记

### 常见错误
- ❌ 在横盘市使用趋势跟踪指标
- ❌ 忽略假信号
- ❌ 过度优化参数
- ❌ 忽视风险/回报比
