## 🔄 自动学习工作流

### 每日学习 (8:00)
```yaml
触发: cron "0 8 * * *"
任务:
  1. 读取每日技术文章 (via web_fetch)
  2. 更新 memory/knowledge/
  3. 生成学习报告
输出: workspace/daily-learning.md
```

### 每周深度学习 (周日 10:00)
```yaml
触发: cron "0 10 * * 0"
任务:
  1. 选择一个专题深度学习
  2. 整理笔记到 memory/
  3. 实践小项目
输出: memory/weekly/[topic].md
```

### 交互学习 (实时)
```yaml
触发: 每次对话
任务:
  1. 识别问题涉及的知识点
  2. 如果是新知识 → 学习并记录
  3. 如果是已知 → 验证并更新
输出: memory/concepts/[topic].md
```

## 🚀 启用自主学习

### 1. 启用定时学习
```bash
# 添加到 crontab
openclaw cron add daily-learning \
  --schedule "0 8 * * *" \
  --task "执行每日学习工作流" \
  --timezone "Asia/Bangkok"
```

### 2. 启用交互学习
在对话中：
- "学习 [主题]" → 主动学习并记录
- "复习 [主题]" → 查看已有知识
- "进度" → 查看学习进度

### 3. 知识验证
定期提问测试知识掌握程度：
```
"测试我的金融知识"
"出几道编程题"
"逆向分析挑战"
```
